echo → print와 같은 기능

ehco”내용”  

내용 출력

echo “내용”  >> 파일이름  
하면 내용을 파일 안에 집어 넣음(파일이 없다면 생성)

git branch -m main 

브랜치 이름 변경

git remote add origin 원격 리포지토리 주소

git remote -v 

**현재 로컬 저장소에 연결된 원격(remote) 저장소들의 목록과 주소(URL)** 를 보여주는 명령어

git push -u origin main
