git init = .git 폴더 생성 →파일을 git 이 관리함

git status  = 현재 깃이 관리하고 있는 폴더에 상태를 보여줌 → 관리하고 있지 않으면 안나옴

git add . =현재 폴더 모든 파일을 커밋할 준비

git config - - global [user.name](http://user.email) “ID”
